Bem-vindo!

No CSS nós posuímos os seletrorespor ID

  • Seletor por Tipo:Busca elementos por uma tage HTMl específica
  • Seletor por ID(#):Busca elementos através do atributo "id"
  • Seletor por Class(.):Busca elementos através do atributo "class" e pode ser passado mais de uma classe dentro do atributo
  • Seletor Universal(*): Seleciona todos os elementos HTML
  • Seletor de Atributo([chave - valor]) Seleciona elementos que possue um atributo específico em nosso documento HTML e conseguimos também buscar atributos com um valor específico
